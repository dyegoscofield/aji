# Quick Bug Diagnosis

Analise o erro/log abaixo e responda APENAS com:

1. **Categoria:** Runtime / Build / Network / Database / Auth / RAG / Stripe
2. **Causa provável:** 1-2 frases
3. **Arquivo(s) envolvido(s):** Caminho(s) provável(is)
4. **Correção sugerida:** Código ou ação específica
5. **Verificação:** Como confirmar que o fix funcionou
6. **Confiança:** Alta / Média / Baixa

Se a confiança for Baixa, diga o que precisa para investigar melhor.
