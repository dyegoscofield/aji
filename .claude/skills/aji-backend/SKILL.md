# Skill: AJI Backend Resources

Recursos de referência para o agente aji-backend. Carregue sob demanda conforme a tarefa.

## Recursos Disponíveis

| Recurso | Arquivo | Quando Carregar |
|---------|---------|-----------------|
| Autenticação e CNPJ | `auth-cnpj.md` | Ao implementar cadastro, login, validação de CNPJ, JWT |
| Stripe e Billing | `stripe-billing.md` | Ao implementar pagamentos, assinaturas, webhooks, comissões |
| Celery Workers | `celery-workers.md` | Ao implementar background tasks, ingestão assíncrona, relatórios |
| Padrões de API | `api-patterns.md` | Ao criar novos endpoints, schemas, respostas de erro |

## Regra de Carregamento

Leia apenas o recurso necessário para a tarefa atual. Não carregue todos de uma vez.
